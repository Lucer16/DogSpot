# Dog Spot | Experiencia Premium para Mascotas 🐾

Dog Spot es una plataforma web de alta gama diseñada para una de las franquicias de cuidado canino más importantes de Panamá. El proyecto se centra en ofrecer una interfaz visualmente impactante, optimizada para la conversión y con un diseño ultra-responsivo.

## 🚀 Tecnologías Utilizadas

* **HTML5 & CSS3** (Estructura semántica)
* **Tailwind CSS** (Framework de utilidades para un diseño moderno y rápido)
* **JavaScript (ES6+)** (Interactividad y lógica de navegación)
* **Swiper.js** (Sliders táctiles de alto rendimiento)
* **Font Awesome** (Librería de iconos vectoriales)
* **Google Fonts** (Tipografía 'Plus Jakarta Sans' para un look tech/moderno)

## ✨ Características Principales

* **Diseño Glassmorphism:** Menús y tarjetas con efectos de desenfoque y transparencia.
* **Arquitectura Bento Grid:** Organización de servicios inspirada en las interfaces de Apple, ideal para visualización en móviles.
* **Secciones Especializadas:**
    * **Pool Pawty:** Landing dinámica para el servicio de piscina.
    * **Dog Spot Express:** Sistema de cotización para servicios a domicilio.
    * **Multi-Sede:** Gestión de información para sucursales (Marbella, Vía Porras, Condado del Rey, Costa del Este).
* **Optimización SEO Avanzada:** Implementación de JSON-LD (Schema.org) para mejorar el posicionamiento en buscadores.
* **Navegación Smooth Scroll:** Experiencia de usuario fluida y sin saltos bruscos.
